# QR-Code-generator

A QR code generator built with HTML, CSS and JavaScript using the QR Server API.

## Features
- Enter any text or URL
- Instantly generates a scannable QR code
- Clean, responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- [QR Server API](https://goqr.me/api/)

## How to Use
1. Enter any text or URL in the input field
2. Click "Generate QR Code"
3. Scan the generated QR code
